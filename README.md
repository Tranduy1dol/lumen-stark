# lumen-stark

[![CI](https://github.com/Tranduy1dol/lumen-stark/actions/workflows/ci.yml/badge.svg)](https://github.com/Tranduy1dol/lumen-stark/actions/workflows/ci.yml)

> ✨ Part of the [luminescent](https://github.com/Tranduy1dol/luminescent) project — *Illuminating the path to zero-knowledge*

A STARK (Scalable Transparent ARgument of Knowledge) implementation following [STARK Anatomy](https://aszepieniec.github.io/stark-anatomy/).

> ⚠️ **Educational Purpose Only**: This library is for learning purposes and is **NOT production-ready**.

## Features

- FRI commitment scheme
- Algebraic Intermediate Representation (AIR)
- STARK prover and verifier (in development)

## Related Projects

This library is part of the [luminescent](https://github.com/Tranduy1dol/luminescent) learning monorepo for Zero-Knowledge Proofs.
